# Palabra de vida

Archivos en este proyecto:

- `index.html` — página principal con botones a tus redes (Instagram, TikTok, YouTube).
- `styles.css` — estilos, paleta dorada/azul y diseño responsivo.

Nota: El sitio está configurado para mostrar Instagram, TikTok y YouTube. El diseño usa tonos dorados y azul profundo para un aspecto cristiano.

Cómo usar:

1. Abre `index.html` en tu navegador.
2. Edita los enlaces dentro de `index.html` para poner tus URLs reales de redes (Instagram, TikTok y YouTube).

Avatar / foto:

- Coloca la imagen que quieras usar en `assets/avatar.jpg` dentro de la carpeta del proyecto. Si prefieres otro nombre o ruta, edita el atributo `src` de la etiqueta `<img id="avatarImg">` en `index.html`.
- Puedes usar una URL remota (por ejemplo `https://.../mi-foto.jpg`) si lo prefieres.

```

python -m http.server 8000

# o con Node (si tienes http-server instalado)
npx http-server -c-1
```

Si quieres, puedo:
- Poner tus enlaces reales ahora.

- Cambiar texto por un versículo o agregar una sección "Acerca de".
